Network Theory & Simulation

Core Networking Concepts

This project integrates several fundamental computer networking concepts to demonstrate a robust enterprise network design. Key areas include:

  • TCP/IP Model: Understanding the layers and protocols governing data transmission.
  • Subnetting & VLANs: Efficient IP address allocation and network segmentation for security and performance.
  • Routing Protocols (OSPF): Dynamic path determination for inter-network communication, ensuring efficient data delivery across complex topologies.
  • Firewall Implementation: Both host-based and network-based firewalls for ingress/egress filtering and access control.
  • Wireless Networking: Deployment and management of Wireless LAN Controllers (WLC) and Lightweight Access Points (LAPs) for secure wireless access.
  • DHCP & DNS: Automatic IP address assignment and domain name resolution for seamless service access.

Simulation Features

The simulated environment allows for comprehensive testing and validation of the network design:

  • Ping, Web Access, DNS and DHCP Testing: Verifying basic connectivity and service functionality across different network segments.
  • Firewall Filtering and ACL Scenarios: Testing the effectiveness of security policies and access control lists in blocking/allowing traffic.
  • Inter-VLAN Routing and Wireless Client Behavior: Observing how devices communicate between different VLANs and how wireless clients obtain network access and roam within the wireless infrastructure.
